# 黑神话·三藏 - 文字引导性游戏

## 项目概述
基于《黑神话·三藏》世界观的轮回主题文字冒险游戏

## 技术栈
- C# .NET 6.0+ 控制台程序
- 纯文本UI界面

## 核心机制
1. **业力-慈悲双轴系统** - 影响战斗形态和结局
2. **因果罗盘系统** - 扫描物件触发前世记忆
3. **蝴蝶效应追踪** - 选择影响后续章节
4. **多周目继承** - 二周目解锁隐藏内容

## 章节结构
- 第一章：轮回之始
- 第二章：西行歧路
- 第三章：佛魔试炼
- 第四章：情镜劫波
- 第五章：灵山真相与因果终局

## 四种结局
1. 梵天涅槃
2. 永劫轮回
3. 三界共主
4. 无间行者

## 运行方式
```
dotnet run
```

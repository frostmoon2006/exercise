# 五子棋游戏 (Gomoku)

一个使用C语言编写的五子棋游戏，支持双人对战和人机对战模式。

## 功能特性

- 🎮 双人模式：两位玩家轮流对弈
- 🤖 人机模式：与AI对战
- 📊 简单的AI算法（评分系统）
- 🔄 完整胜负判定（横、竖、斜四个方向）
- 🎯 胜利连线高亮显示

## 编译运行

```bash
gcc gomoku.c -o gomoku -std=c99
./gomoku.exe
```

## 游戏规则

1. 15×15 的棋盘
2. 黑白双方轮流落子
3. 先连成5子获胜（横/竖/斜）
4. 棋盘满则平局

## 控制说明

- 输入坐标格式：`列 行`（如 `8 8` 表示中心位置）
- 输入 `q` 退出游戏
- 输入 `r` 重新开始
